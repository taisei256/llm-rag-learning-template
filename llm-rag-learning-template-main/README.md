# LLM・RAG 学習システム

## 🎯 このプロジェクトについて

このテンプレートは、LLM・RAG 技術を学習するための基本構造です。

## 📁 プロジェクト構造

- `js/` - JavaScript ファイル
  - `config.js` - API 設定（あなたの学生 ID を設定）
  - `llm-client.js` - LLM API 通信用クラス
  - `app.js` - メインアプリケーション
- `css/` - スタイルシート
- `data/` - サンプルデータ
- `docs/` - ドキュメント

## 🚀 セットアップ

### 1. このリポジトリをフォーク

GitHub で「Fork」ボタンをクリック

### 2. 自分の PC にクローン

```bash
git clone https://github.com/あなたのユーザー名/llm-rag-learning-template.git
cd llm-rag-learning-template
```
